한 줄에 한 명씩, 1번부터 순서대로 입력하거나 명렬표를 복사해 붙여넣으세요. 비워두면 번호만 표시됩니다.

💾 데이터

자동 저장은 지금 사용하는 컴퓨터의 브라우저에 저장되는 기능입니다.
다른 컴퓨터에서 이어서 쓰려면 우리 반 설정 저장하기로 .json 파일을 만들고, 다른 컴퓨터에서 이 HTML 파일을 연 뒤 저장한 우리 반 설정 가져오기를 누르면 됩니다.
HTML 파일은 프로그램, JSON 파일은 우리 반 설정 저장 파일이라고 생각하시면 됩니다.

🤫 선생님 전용 설정

학생들에게는 보이지 않는 비밀 기능입니다. 여러 명 조건 설정도 가능합니다. (오른쪽 아래 🐾 2번 연속 터치 / Ctrl+Shift+T)
배치 조건 없음

🐾 배치 조건

모든 조건은 이 패널에만 표시됩니다. 학생 화면의 책상·상자·뽑기 연출에는 라벨이 나타나지 않습니다.

📌 지정 좌석

좌석에 학생을 지정하면 뽑기에서 반드시 그 자리로 갑니다. 연출은 똑같아서 티가 나지 않아요. 모든 좌석을 지정하면 완전 수동 배치가 됩니다.
🪑 후보 구역 좌석을 클릭하세요. 0개 선택됨

⚙️ 선생님 메뉴

학생에게 보여줄 필요 없는 기능을 이곳에 모았습니다. 평소에는 첫 화면을 단순하게 사용할 수 있어요.
배치 조건은 앞자리, 구역 배치, 멀리/가까이 앉히기, 지정 좌석을 설정하는 선생님 전용 기능입니다.

❓ 고양이 자리 뽑기 사용법

처음 쓰는 선생님도 아래 3단계만 따라 하면 바로 사용할 수 있어요.

✅ 처음 사용 3단계

① ⚙️ 선생님 메뉴 → ⚙️ 학급 설정
학생 수, 이름, 결번을 입력합니다.

② 필요하면 ⚙️ 선생님 메뉴 → 🪑 책상 배치
책상을 드래그하거나 기본/짝꿍/모둠/ㄷ자형으로 바꿉니다.

③ 📦 한 자리 뽑기 또는 ⚡ 전체 자동
한 명씩 뽑거나 전체를 자동으로 배치합니다.

🎲 자리 뽑기 버튼

• 📦 한 자리 뽑기: 한 명씩 천천히 뽑습니다.
• ⚡ 전체 자동: 전체 학생을 자동으로 배치합니다.
• 🔄 다시 하기: 처음부터 다시 뽑습니다.

⚙️ 선생님 메뉴

첫 화면이 복잡하지 않도록 세부 기능은 ⚙️ 선생님 메뉴 안에 넣었습니다.
• 학급 설정
• 책상 배치
• 좌석표 출력/저장
• 전체화면
• 음악 켜기/끄기
• 배치 조건

🤫 배치 조건

학생 화면에는 조건이 절대 표시되지 않습니다.
뽑기 연출도 조건이 있든 없든 똑같이 보입니다.

• 지정 좌석: 특정 학생을 특정 자리에 고정
• 앞자리 배치: 앞 1~2줄 안에서 랜덤 배치
• 구역 배치: 선생님이 고른 좌석 후보 안에서 랜덤 배치
• 멀리 앉히기: 가까이 앉으면 안 되는 학생들을 떨어뜨림
• 가까이 앉히기: 도우미 친구처럼 가까이 앉혀야 하는 학생들을 묶음
여러 명 선택은 Ctrl 키를 누른 채 클릭하면 됩니다.

💾 다른 컴퓨터에서 이어서 쓰기

HTML 파일 = 프로그램 / JSON 파일 = 우리 반 설정 저장 파일
같은 컴퓨터에서는 자동 저장됩니다.
다른 컴퓨터로 옮길 때는 학급 설정 → 우리 반 설정 저장하기로 JSON 파일을 만들고, 다른 컴퓨터에서 저장한 우리 반 설정 가져오기를 누르면 됩니다.

🖨 좌석표 출력/저장

원하는 형식을 선택하세요.
이미지는 바로 인쇄·공유하기 좋고, 한글 편집용 문서는 선생님이 표를 수정하기 좋습니다.
한글 편집용 문서는 .doc 형식으로 저장됩니다.
