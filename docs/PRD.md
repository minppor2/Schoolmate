# 스쿨메이트 AI
## 학교별 맞춤형 교사 업무 자동화 DIY 플랫폼

---

## 1. 제품 개요

### 1.1 제품명

**스쿨메이트 AI**

### 1.2 한 줄 설명

Google Chat 업무방과 연동하여 교사의 업무 메시지를 AI가 자동으로 분석하고, 교사가 확인 후 업무카드·일정·알림·공지문으로 전환할 수 있게 돕는 학교 맞춤형 교사 업무 자동화 웹앱

### 1.3 핵심 콘셉트

스쿨메이트 AI는 학교 업무 전체를 한 번에 해결하는 거대한 시스템이 아니다.

MVP의 핵심은 다음 한 가지 흐름을 완성하는 것이다.

```text
Google Chat 업무 메시지
→ AI 업무 후보 추출
→ 교사 확인
→ 업무카드 저장 / 일정 등록 / 공지문 변환
```

이 핵심 흐름을 중심으로 일정 관리, 특별실 예약, 학생기록 도우미, 학교 DIY 설정을 단계적으로 확장한다.

---

## 2. 문제 정의

### 2.1 교사가 겪는 실제 문제

학교 업무는 여러 채널에 흩어져 있고, 교사는 같은 내용을 반복해서 정리해야 한다.

- Google Chat, 문자, 메일, 학교 메신저 등으로 업무가 분산됨
- 중요한 일정이 메시지 속에 묻힘
- 메시지를 보고 일정표, 업무 목록, 공지문에 다시 입력해야 함
- 특별실 예약 현황을 별도로 확인해야 함
- 생기부·세특·학생기록 관련 자료와 지침 파일이 흩어져 있음
- 복사·붙여넣기 기반 AI 도구는 교사에게 또 다른 입력 업무가 됨

### 2.2 핵심 문제

> 교사는 업무 메시지를 확인하는 것에서 끝나지 않고, 그 메시지를 다시 일정·업무·공지·기록으로 바꾸는 반복 작업까지 해야 한다.

## 3. 제품 목표

### 3.1 핵심 목표

교사가 Google Chat에서 평소처럼 업무 메시지를 주고받으면, 스쿨메이트 AI가 승인된 업무방의 메시지를 불러와 AI가 업무 후보를 추출하고, 교사가 확인 후 업무카드와 일정으로 전환할 수 있도록 한다.

### 3.2 사용성 목표

- 복사·붙여넣기를 최소화한다.
- 첫 화면에는 꼭 필요한 정보만 보여준다.
- AI는 자동 실행하지 않고 후보만 만든다.
- 교사가 확인한 내용만 저장·등록된다.
- PC와 모바일에서 각각 다른 화면 구조를 제공한다.
- 사용자는 보기 모드를 선택할 수 있다.
- 학교별 기본 템플릿을 제공하여 초기 설정 부담을 줄인다.

## 4. 핵심 MVP 범위

### 4.1 MVP 핵심 기능

MVP는 아래 기능에 집중한다.

| 우선순위 | 기능 | 설명 |
|---:|---|---|
| 1 | 로그인 | Google 계정 기반 사용자 인증 |
| 2 | 홈 화면 간단 보기 | 오늘 할 일, AI 업무 후보, 빠른 실행 |
| 3 | Google Chat 메시지 불러오기 | 승인된 업무용 Space 메시지 조회 |
| 4 | AI 업무 후보 생성 | 메시지에서 날짜, 대상, 할 일 추출 |
| 5 | 업무카드 저장 | 교사가 확인한 업무만 DB 저장 |
| 6 | 일정 등록·수정·삭제 | 일정 관리 및 Google Calendar 연동 |
| 7 | 알림 배지/팝업 | 오늘 일정, 중요 일정, 새 업무 후보 표시 |

### 4.2 MVP에서 작게 구현할 기능

| 기능 | 축소 구현 방향 |
|---|---|
| 특별실 예약 | 날짜·교시·실명 기준 예약과 중복 방지 |
| 학생기록 자료실 | 링크와 파일 저장 중심 |
| 세특 점검 | 샘플 문구 점검 중심, 작성 대체 금지 |
| 학교 DIY 설정 | 중학교 기본 템플릿 제공 후 일부 수정 |

### 4.3 2차 확장 기능

| 기능 | 확장 방향 |
|---|---|
| Google Chat 실시간 이벤트 감지 | Workspace Events API 활용 |
| 모바일 푸시 알림 | Firebase Cloud Messaging 활용 |
| 반복 예약 | 특별실 예약 고도화 |
| Chat 메시지 버튼 삽입 | Google Chat App 확장 |
| 관리자 통계 | 업무 처리량, 일정 누락 감소 확인 |
| 학교별 템플릿 공유 | 다른 학교 적용성 강화 |

## 6. 핵심 차별점

### 6.1 노션과의 차이

| 구분 | 노션 | 스쿨메이트 AI |
|---|---|---|
| 성격 | 범용 문서·DB 도구 | 학교 업무 특화 자동화 플랫폼 |
| 초기 설정 | 사용자가 직접 설계 | 학교급별 기본 템플릿 제공 |
| 메시지 처리 | 직접 입력 또는 복사 필요 | Google Chat 업무방 연동 |
| AI 활용 | 별도 설정 필요 | 업무 후보 추출이 핵심 기능 |
| 일정 등록 | 수동 입력 중심 | 메시지에서 일정 후보 추출 |
| 특별실 예약 | 직접 표 제작 필요 | 교시·장소 기반 중복 예약 방지 |
| 학생기록 | 자료 저장 수준 | 자료실 + 지침 파일 + 세특 점검 보조 |
| 화면 구성 | 사용자가 직접 구성 | 교사별 보기 모드와 위젯 설정 |

### 6.2 스쿨메이트의 차별화 문장

> 스쿨메이트 AI는 노션처럼 교사가 직접 구조를 만드는 도구가 아니라, 학교 업무 흐름을 미리 반영한 상태에서 Google Chat 메시지를 업무와 일정으로 자동 전환하는 교사용 업무 자동화 플랫폼이다.

## 7. 핵심 사용자 흐름

### 7.1 Google Chat 업무 자동 정리 흐름

```text
1. 교사가 Google Chat 업무방에서 평소처럼 메시지를 주고받는다.
2. 스쿨메이트가 관리자가 승인한 Space의 메시지만 불러온다.
3. AI가 업무성 메시지를 분석하여 업무 후보를 만든다.
4. 홈 화면에 AI 업무 후보가 표시된다.
5. 교사가 내용을 확인한다.
6. 교사가 업무카드 저장, 일정 등록, 공지문 변환 중 선택한다.
7. Firestore와 Google Calendar에 반영된다.
```

### 7.2 일정 등록 흐름

```text
1. AI가 메시지에서 날짜와 시간을 추출한다.
2. 날짜가 불분명하면 "확인 필요"로 표시한다.
3. 교사가 제목, 날짜, 시간, 대상, 장소를 확인한다.
4. 교사가 일정 등록을 누른다.
5. Firestore에 일정이 저장된다.
6. Google Calendar에 등록된다.
7. syncStatus가 synced로 변경된다.
```

## 8. 화면 구성 원칙

### 8.1 첫 화면 표시 원칙

홈 화면에는 아래 3가지만 표시한다.

1. 오늘 할 일 최대 3개
2. AI 업무 후보 최대 2개
3. 빠른 실행 버튼 최대 3개

나머지 기능은 메뉴 안으로 숨긴다.

### 8.2 숨겨야 할 기능

- 전체 특별실 예약표
- 월간 대형 달력
- 관리자 설정
- 권한 관리
- Google Chat 세부 설정
- 학생기록 전체 자료 목록
- AI 분석 상세 로그
- 2차 확장 기능

## 9. 사용자별 보기 모드

| 모드 | 설명 |
|---|---|
| 자동 맞춤 | 화면 크기에 따라 PC/모바일 구조 자동 변경 |
| PC 보기 | 넓은 대시보드형 화면 |
| 모바일 보기 | 1열 카드와 하단 탭 중심 화면 |
| 간단 보기 | 오늘 할 일, AI 업무 후보, 빠른 실행만 표시 |
| 관리자 보기 | 학교 설정, 권한, 연동 관리 표시 |

기본값은 **자동 맞춤**이다.  
일반 사용자에게는 복잡한 모드 선택을 강요하지 않고, 홈 화면에 **간단 보기 전환**만 노출한다.

## 10. 개인정보 및 안전 원칙

### 10.1 Google Chat 데이터

- 모든 대화를 수집하지 않는다.
- 관리자가 승인한 업무용 Space만 연동한다.
- 개인 DM은 수집하지 않는다.
- 민감 정보가 포함될 수 있는 메시지는 분석 전 경고한다.
- 메시지 원문은 보관 기간을 설정할 수 있다.

### 10.2 학생기록 데이터

- 실제 학생 개인정보 입력을 최소화한다.
- 발표 및 시연에서는 가상 데이터를 사용한다.
- 학생기록 도우미는 생기부 작성 대체 기능이 아니다.
- AI는 교사가 작성한 문구를 점검하고 개선 의견을 제시하는 보조 도구다.
- 원문 저장 여부를 사용자가 선택할 수 있게 한다.

### 10.3 AI 안전 원칙

- AI는 후보만 생성한다.
- AI 결과는 자동 실행하지 않는다.
- 교사가 확인한 후에만 업무 저장, 일정 등록, 공지문 생성이 가능하다.
- 신뢰도 낮은 결과는 “확인 필요”로 표시한다.

## 12. 발표용 핵심 문장

복사·붙여넣기 방식은 교사에게 또 다른 업무가 될 수 있다고 판단했습니다.  
그래서 스쿨메이트 AI는 Google Chat 업무방과 연동하여 교사가 평소처럼 메시지를 주고받기만 해도 AI가 업무와 일정을 후보로 추출하도록 설계했습니다.  
단, 모든 대화를 수집하지 않고 관리자가 승인한 업무용 Space만 연동하며, AI가 자동 실행하지 않고 교사가 확인한 뒤 업무카드와 일정으로 저장하는 안전한 구조를 적용했습니다.

## 13. 최종 제품 설명

스쿨메이트 AI는 학교별 업무 방식에 맞게 설정할 수 있는 교사용 업무 자동화 플랫폼이다.  
Google Chat 업무방과 연동하여 메시지 속 업무와 일정을 AI가 후보로 추출하고, 교사는 확인 후 업무카드, 일정, 공지문, 알림으로 전환할 수 있다.  
또한 특별실 예약, 학생기록 자료실, 세특 문구 점검 보조 기능을 제공하여 교사의 반복 행정 업무를 줄이고 학교 업무를 한곳에서 관리할 수 있도록 돕는다.
